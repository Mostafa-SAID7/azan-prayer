# 🚀 Quick Start - 5 Minutes

Get Payer running in 5 minutes.

## Prerequisites

- Node.js 18+ ([Download](https://nodejs.org))
- Git
- Terminal/Command line

## Steps

### 1. Clone Repository (1 min)

```bash
git clone https://github.com/yourusername/payer.git
cd payer
```

### 2. Install Dependencies (2 min)

```bash
npm install
```

### 3. Start Dev Server (30 sec)

```bash
npm run dev
```

Output:
```
VITE v5.4.1  ready in 123 ms

➜  Local:   http://localhost:5173/
➜  press h to show help
```

### 4. Open in Browser (30 sec)

Click or copy the local URL: `http://localhost:5173`

### 5. Try Features (1 min)

- 📍 Enter a city name
- 🌙 Toggle dark mode
- 🗣️ Switch language (Arabic/English)
- 📱 Check responsive design (F12 → Toggle device toolbar)

## What's Working?

✅ Prayer times display
✅ Dark mode toggle
✅ Language switcher
✅ Responsive design
✅ Offline capability (PWA)
